## Run

node > 16.0

### Install

```sh
pnpm install
```

### Develop

```sh
pnpm run dev
```

### Build

```sh
pnpm run build
```

## commit

use: husky + commitlint

- method 1: use cli

```shell
pnpm run commit
```

- method 2: use git

```shell
git commit -m "feat: add a new feature"
```
